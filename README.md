# Trustees — The Middle Man You Can Trust

Trustees is a Consumer-to-Consumer (C2C) e-commerce platform 
built for the South African informal economy. Unlike traditional 
online marketplaces, Trustees combines a digital platform with a 
physical location to provide a safe and trusted trading environment 
for everyday buyers and sellers.

## How it works

1. A seller lists their product on the Trustees platform
2. The seller drops the item off at the nearest Trustees location
3. A Trustees staff member inspects and verifies the item
4. The buyer is notified that the item is ready
5. The buyer can collect at the Trustees location or choose delivery

## Features

- User registration and login
- Identity verification for buyers and sellers
- Product listings with image uploads
- Simulated wallet and payment system
- Order tracking with status updates
- Buyer protection and dispute resolution
- Admin dashboard with Role-Based Access Control (RBAC)

## Tech Stack

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL
- Version Control: Git & GitHub
- Local Development: XAMPP

## Database Setup

To set up the database: in phpMyAdmin, import `database/trustees_db.sql`, then import `database/seed.sql`.
